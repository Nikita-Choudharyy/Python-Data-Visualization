# 📊 Matplotlib

Welcome to the **Matplotlib** section of the **Python Data Visualization** repository.

This folder provides a structured learning path to master Matplotlib, one of the most widely used Python libraries for creating static, animated, and interactive visualizations.

Whether you're a beginner learning the fundamentals or an aspiring data scientist building real-world projects, these notebooks will help you develop strong data visualization skills.

---

# 📚 Learning Path

Follow the notebooks in the order below:

## 1️⃣ 01_Basics.ipynb

Learn the core concepts of Matplotlib, including:

* Introduction to Matplotlib
* Creating line plots
* Figure size
* Titles and axis labels
* Colors
* Line styles
* Markers
* Grid
* Legends
* Multiple line plots

---

## 2️⃣ 02_Customization.ipynb

Take your visualizations to the next level by learning:

* Figure and Axes objects
* Multiple subplots
* Plot styles
* Custom colors
* Colormaps
* Tick customization
* Legends
* Annotations
* Shapes
* Saving figures

---

## 3️⃣ 03_Advanced.ipynb

Explore advanced Matplotlib capabilities, including:

* GridSpec layouts
* Twin axes
* Logarithmic scales
* Error bars
* Stem plots
* Step plots
* Fill-between plots
* Stack plots
* Event plots
* Spy plots
* Date-based visualizations
